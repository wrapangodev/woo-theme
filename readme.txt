== Description ==
Cozynest is a fully responsive WordPress FSE theme that will bring your electronics store online. Fully integrated with WooCommerce, you get a solution that makes selling things online.
The theme includes pre-design patterns and full-page templates that allow you to give unique look to your site as you need. You can start building your content with the Drag-and Drop interface using the blocks.
It lets you change individual element colors and border colors to create just the look you want. As it supports full site editing, you can easily preview the layout on mobile and tablet from the Site Editor.
Whether you’re building a single-page website, a blog, a business website, or a portfolio, Cozynest will help you create a site that is uniquely yours.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Cozynest in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
5. Navigate to Appearance > Editor in your admin panel and customize to taste.
